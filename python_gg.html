Go Back Python Game #1

Guessing Game

This project was my first game that I programmed using Python. It was a project in our Progamming Logic and Design course. This project was our first take on using tkinter to make graphical user Interface (GUI) For the elements, I used Adobe Photoshop. This was also my first attempt in using Photoshop to create buttons, backgrounds and other elemets.

Mechanics of the game

The game will let you guess three numbers from one (1) to nine (9). If you guessed a number, you will win a prize. The amount of the prize depends on the number you have guessed.
