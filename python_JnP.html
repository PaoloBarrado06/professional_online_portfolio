Jack en Poy

This is my second output for our GUI game project. This is also a python programmed game using tkinter for the graphical user intergace (GUI). For the elements, I used Adobe Photoshop to create and edit

Mechanics of the game

Jack en Poy is the Filipino term for "Rock, Paper and Scissors". They both have the same rules. In this game, the player will choose what hand to play then the player will press reveal to see what hand will the computer play.
